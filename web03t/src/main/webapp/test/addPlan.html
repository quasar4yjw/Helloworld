여행지 선택

  • 여행하고 싶은 국가를 선택해주세요.

언어 선택

  • 가이드할 언어를 선택해주세요.

제목 선정

  • 여행아이템 리스트에 노출되는 항목입니다.
  • 여행의 특징이 잘 드러나도록 제목을 정해주세요.

가격

  • 관광 상품의 가격을 정해주세요.
  • 가격은 6자리까지 가능합니다.

관심사 선택

  • 여행의 주요 테마를 세 가지 선택해주세요.
  • 적어도 하나 이상을 선택해주세요.

주요 관광 장소

  • 여행아이템 리스트에 노출되는 항목입니다.
  • 대표 관광 장소를 적어주세요.

일차, 시간, 여행 상세 내용

  • 관광객들에게 여행 일정을 알려주세요.
